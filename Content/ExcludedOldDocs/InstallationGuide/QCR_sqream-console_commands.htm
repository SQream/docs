Quick Command Reference

sqream-console commands

Use these console commands to install and manage SQream.

sqream install

Installs the sqream docker image in the specified location.

Use this command to create or modify the storage cluster.

Available flags:

--image: installs sqream docker image on local host docker registry.

--registry: installs sqream docker image on server docker registry.

--volume

--create: create new sqream storage directory.

--mount: mount to an existing sqream storage directory.

--upgrade: upgrade sqream storage.

--help: display available flags

sqream master

Starts the sqream master server container (metadata server and server picker).

Use this command also to stop the server.

Available flags:

--start|stop: starts/stops metadata server on port 3105, server picker on port 3108.

--mode

--single-host/distributed

--help: display available flags

sqream worker

Starts the sqream worker container.

Use this command to do the following:

  • start workers and assign them ports and allocate memory
  • list all running workers
  • for a specific worker (indicated by the port flag):
    • stop the worker
    • allocate a specific GPU index
    • modify memory allocation
Available flags:

--start: number of sqream workers to start.

--stop: stop a specific sqream worker.

--list: get a listing of all running workers.

--gpu: allocate specific GPU index for this worker.

--ports: port assigned to worker, default is 5000.

--memory: limit spool memory, GB.

--help: display available flags

sqream help

Displays all available sqream commands.
