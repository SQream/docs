Basic Docker commands

Use these Docker commands to find containers and images.

List all running containers (names, ID's etc.):

docker ps

Log inside a running container:

docker exec -it <container-name or container-id> bash

List all images installed on machine:

docker images

Start a new container running on the same network (all sqream running containers are available by name via DNS):

docker run -it --rm --network sqream_network <sqream-image-name:tag> bash
