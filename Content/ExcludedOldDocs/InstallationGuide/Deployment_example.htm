Deployment example

Here is a sample of commands used to deploy a single-host deployment with 3 workers.

single-host deployment

  1. Create new sqream storage on $(pwd)/data local folder:

    2. sqream install --volume –-create $(pwd)/data

  2. Start sqream master in single-host mode:

    3. sqream master --start --single-host

  3. Start 3 sqream workers on host, listening on ports 5000, 5001, 5002, each one with 100 GB of memory allocated:

    4. sqream worker --start 3 -p 5000 --memory 100
