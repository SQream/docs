sqream-console configuration file

sqream.ini is the configuration file that holds all global variables for the current console session.

Variables to be set by the user

Some configuration variables are automatically updated, some are hard-coded, and some can optionally be set by the user.

Note that the following 2 variables must set by the user.

  • Set host_ip to the public ip address of your machine.
  • Set mac_address to the mac address for which your license was created.

Configuration file variables

The following section from the sqream-ini configuration file lists the env variables group .

Note the highlighted variables that you must set.
