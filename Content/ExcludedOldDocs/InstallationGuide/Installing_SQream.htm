Installing SQream

The steps below present the general forms of the sqream commands used for installation. For a complete description of the commands see Quick Command Reference sqream-console commands.

  1. Install the SQream console

    2. sqream-console is a management tool that allows you to install, set up, and run, a SQream cluster.

    To install sqream-console:

    1. Copy sqream-console from device to host:

      2. cp -r <folder path on device> <destination path on host>

    2. Navigate to sqream-console directory:

      3. cd sqream-console

    3. Run the following command to install the sqream-console-install application:

      4. ./sqream-console-install.sh

  2. Run the SQream console

    3. Once the console is successfully installed, run the following executable bash to start the sqream-console application:

    ./sqream-console

    You are now ready to install and set up your SQream cluster using the tools provided by the console.

  3. Install SQream

    4. Run the sqream install command to install SQream and create the storage cluster.

    sqream install --volume --create <location>

  4. Start the sqream master

    5. Run the sqream master command to start the sqream master in the specified mode.

    sqream master --start --<mode>

  5. Start the sqream workers

    6. Run the sqream worker command to start the sqream workers on the specified ports.

    Default port is 5000. For multiple workers, the specified port will be assigned to the first worker and successive ports to the remaining workers.

    You can also allocate a specified amount of memory to each worker.

    sqream worker --start <number of workers> -p <port> --memory <GB of memory>

    You are now ready to connect SQream using the SQream command line. See Connect to the SQream DB server with clientcmd.
    For other connecters, please see Drivers.
