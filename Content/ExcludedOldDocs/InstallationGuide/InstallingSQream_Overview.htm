Installing SQream using Docker

Overview

SQream uses Docker to quickly and easily install, set up, and run SQream from Docker containers. Containers include the minimal runtime requirements of the application, to allow you to install and deploy SQream without the necessity of setting up the environment

Make sure that you have completed all the procedures described in the' SQream Preinstallation Procedures' manual and that Docker is properly installed.

This document describes how to install the SQream console, and then use the console to install SQream. It also includes a short command reference of sqream and Docker commands.
