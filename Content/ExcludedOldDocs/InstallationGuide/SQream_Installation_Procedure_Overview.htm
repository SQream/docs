SQream Installation Procedure Overview

These are the steps for installing SQream:

  1. Install the SQream console application (Install the SQream console)
  2. Run the console (Run the SQream console)
  3. Install SQream and allocate storage (sqream install) (Install SQream)
  4. Start the master (sqream master) (Start the sqream master)
  5. Start the workers (sqream worker) (Start the sqream workers)

SQream Installation and Management Tools

sqream-console commands:

  • sqream install: Installs sqream and creates or modifies sqream storage
  • sqream master: manages sqream master
  • sqream worker: manages sqream workers

sqream-console configuration file

sqream.ini is the sqream-console configuration file that holds all global constants for the current session.
