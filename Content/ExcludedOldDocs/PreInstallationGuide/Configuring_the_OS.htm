Configuring the OS

The following OS configuration steps are required as part of the basic setup (with the exception of the optional utilities). This will increase the security and usability of your server and provide a solid foundation for optimal SQream performance.

  • Making sure Development Tools and Debugging Tools are installed
  • Installing useful utilities (Optional)
  • Configuring the OS language and timezone
  • Tuning OS parameters
  • Configuring the network

To begin the OS configuration:

  • Log on to the server as "root" user.
