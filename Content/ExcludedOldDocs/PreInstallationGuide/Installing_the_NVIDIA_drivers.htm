  2. Run the following command:

    3. nvidia-smi

  3. Reboot

Troubleshooting the NVIDIA driver

Sometimes, the NVIDIA driver will fail to install if SELINUX is enabled. SELINUX tends to interfere with NVIDIA driver installation and should be disabled by editing the /etc/selinux/config configuration file.

To see whether SELINUX is enabled, run the following command:

# sudo sestatus

If SELINUX enabled you can either disable it temporarily, for the current session only, or permanently (recommended).

To temporarily disable SELINUX, run the following command:

# sudo setenforce 0

To permanently disable SELINUX (recommended):

  1. Run the following command to edit the /etc/selinux/config configuration file:

    2. sudo vim /etc/selinux/config

  2. Change the line that reads SELINUX=enabled to:

    3. SELINUX=disabled

  3. Reboot
