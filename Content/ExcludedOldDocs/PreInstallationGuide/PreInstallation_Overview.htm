SQream Preinstallation Procedures

Overview

The SQream installation process has been carefully structured to ensure maximum compatibility and reliability. It contains some logging steps that can provide troubleshooting information if needed.

There are a number of preliminary steps that you must perform before you can actually install SQream:

  • Configuring the OS
  • Installing NVIDIA drivers
  • Installing the NVIDIA-Docker and Docker

The installation process must be carried out on each server separately.
