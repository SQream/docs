Statement

Table 2. Statement execution
Function Description

StatementHandle(ConnectionHandle, 'statement')

Creates a statement object which purpose is to operate the different messages of the protocol in order to execute a query and its different functionalities.

StatementHandle.prepare()

Prepares the statement of the current StatementHandle.

StatementHandle.execute()

Executes the statement of the current StatementHandle. Comes after prepare().

StatementHandle.nextRow()

On an insert query - start setting the next row for insertion. SQream does not support partial inserts. On a select query - move to next row index to start selecting items from various columns using get() functions

StatementHandle.close()

Closes the StatementHandle.
