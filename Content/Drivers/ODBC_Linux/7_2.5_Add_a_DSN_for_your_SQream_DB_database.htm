Add a DSN for your SQream DB database

Edit the odbc.ini file and add the ODBC DSN information.

Verify that the odbc.ini file contains the following lines:

[{DSN name}]  
Driver={path of sqream_odbc.so}  
Description=sqream64
Server={IP address of the SQream server}  
Port={Listener port of SQream}  
Database={SQream database to connect to}  
User={SQream DB user}  
Password={password of the above SQream DB user}  
Service={service to use when connecting}  
Cluster={true  | false}  
ssl={true  | false}

For example:

[sqreamOn251]  
Driver=/home/sqream/ODBC/sqream_odbc.so
Description=sqream64
Server=192.168.0.251
Port=3108  
Database=master
User=sqream
Password=sqream
Service=sqream
Cluster=true  
ssl=false
