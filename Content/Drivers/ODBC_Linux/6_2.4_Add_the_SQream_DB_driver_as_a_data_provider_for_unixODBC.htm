Add the SQream DB driver as a data provider for unixODBC

Edit the odbcinst.ini file you found in the previous step.

Verify that the odbcinst.ini file contains the following lines:

[sqream64]  
Description=sqream64
Driver={path of sqream_odbc.so}  
Setup={path of sqream_odbc.so}  

[ODBC]  
Trace={yes|no}  
TraceFile={path and file name of the trace file}

For Example:

[sqream64]  
Description=sqream64
Driver=/home/sqream/ODBC/sqream_odbc.so
Setup=/home/sqream/ODBC/sqream_odbc.so

[ODBC]  
Trace=no
TraceFile=/home/sqream/ODBC/trace.tmp
