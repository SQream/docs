Testing the SQream ODBC Driver

Use the ISQL utility to test the driver:

On the command line, specify the DSN name:

isql [DSN name]

For the above example this would be:

isql SqreamODBC

ISQL is a unixODBC command-line interactive SQL tool that can be used as a tester. No additional installation is required (UnixODBC is a pre-requisite for SQream).

For more information, see https://www.systutorials.com/docs/linux/man/1-isq/l
