Install the SQream ODBC Driver and Configuration

Download the SQream ODBC driver

Download the SQream ODBC driver for Linux from: https://sqream.com/product/client-drivers/

Untar (extract) the sqream_odbc.tar.gz file into any temporary location:

tar -xf	sqream_odbc.tar.gz

The extracted sqream_odbc.so file is the ODBC driver file.
