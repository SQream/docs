Installing JDBC driver using the Windows installer

Download as PDF

Click here to download the Windows installer

  1. Download the SQream driver package from the link above. This will download the latest version.

    If you know your instance is running a different version, please contact the SQream support team for the correct link.

  1. Run the installer and follow the steps in the Wizard.

  1. The User Account Control window may open.

    Click Yes to continue the installation.

  1. Follow the installation instructions.

    Choose the components you wish to install.

    If you have a previous version installed, make sure Remove old JDBC Driver Registry is selected.
