Configuring the SQL Workbench Client

Download as PDF

Recommended method - Fresh install

(Windows x64)

  1. Download the latest installer here: https://sqream.com/product/client-drivers/ (choose JDBC Windows Installer).

    Follow the installation wizard instructions.

    Make sure you check both JDBC and SQL Workbench sections.

  2. You should now have a Start menu item for SQL Workbench.

    Hit the Windows button on the screen or keyboard, and type SQL Workbench. Select either the 64 bit or 32 bit variant, depending on your preference.

  3. Open SQL Workbench. A New Connection window should be displayed with the SQream driver already selected.

  4. Fill in the connection string URL (obtained from SQream).

  5. You are now ready to start querying data!
