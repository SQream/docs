Fixes

  • JDBC 2.9 for Windows now supports UTF-8 Unicode correctly
  • Amount of data read in the ReadTable chunk producer is now correctly reported in the show_node_info utility function
  • Chunk producer elapsed time is now correctly reported in the show_node_info utility function
  • When very large tables are deleted or truncated, a client may have a long delay upon first connection
  • Python 3.X connector compatibility improved (Map() returns an iterator instead of a list)
