SQream DB Version 2019.1.2.1

Release Notes - March 27, 2019

What's New

  • Added support for more functionality with UTF-8-encoded NVARCHAR datatype:
    • GROUP BY aggregations
    • DISTINCT aggregations
    • CAST from ASCII-only VARCHAR to UTF-8-encoded NVARCHAR
  • Added support for TRIM / LTRIM / RTRIM operations on VARCHAR
