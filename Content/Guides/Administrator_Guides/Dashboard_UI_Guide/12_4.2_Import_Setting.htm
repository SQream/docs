Import Setting

Under Import Setting you configure the layout of the CSV file you want to import.

Basic settings include:

  • Delimiter - Specify the correct column delimiter, like ,, \t, |, or any other single character. Comma is the default delimiter.

  • Header line - Check this checkbox to skip the first line of the file.

More advanced settings are accessible via Advanced Settings link. Refer to SQream SQL Reference Guide under the "COPY FROM (bulk import)” section for more information about these settings.
