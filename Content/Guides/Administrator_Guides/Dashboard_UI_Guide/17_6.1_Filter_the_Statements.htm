Filter the Statements

Use the various filter options to narrow down the displayed statements. You can use one or multiple filter options offered in the top part of the screen:

  • Search - Search for a key word in the Query Statement or Error Message.

  • Filter by database - From the dropdown list select a specific database.

  • From …​ to …​ - Use the calender and time icons to set the start and end time of your filter condition.

  • Status - From the dropdown list select a specific status.
