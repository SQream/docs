Select Table and Settings

  • From the dropdown box select the database and respective table.

  • Rows: The number of rows in the table to run the optimization process on. 1,000,000 is the default.

  • Buffer Size: For VARCHAR optimization the system checks the actual data length of the field and adds a buffer to the recommendation. 10% is the default.

  • Optimize NULLs: Select this checkbox to activate the optimization for fields that allow NULL.

Click EXECUTE to start the optimization process.
