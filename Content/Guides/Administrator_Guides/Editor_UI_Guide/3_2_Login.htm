Login

Open your browser and navigate to the URL of your SQream Statement Editor.

This is your login screen:

Figure 1. Login Screen

Log in with your SQream DB credentials, including server IP and port, username and respective password, and database name. It is recommended to connect to the Load Balancer (Server Picker) of the SQream database. Check the respective check box. Then click LOG IN.

If this is your first connection, the master database is your default database.

For AWS cloud users, the default credentials are:
Username: sqream
Password: <Your EC2 Instance ID>.
