Optimizer Results

In the OPTIMIZED COLUMNS tab you can review the system recommendation to (1) decrease the length of VARCHAR fields or (2) remove the allow NULL option. In the OPTIMIZED DDL tab you can review the DDL statement to create the recommended table.

  • COPY DDL TO CLIPBOARD - Copies the DDL statement to create the recommended table to the clipboard

  • CREATE A NEW TABLE - Creates a new table in same database and schema as the original table. New table name = [table_name]_new. No data is populated to the new table.

  • CREATE AND INSERT INTO EXISTING DATA - Create a new table in same database and schema as the original table and populates the data.
