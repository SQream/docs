SQream Installation Guides

Installing SQream

The SQream installation process has been carefully structured to ensure maximum compatibility and reliability.

SQream supports two installation methods:

  • Docker (recommended)
  • RPM (if you cannot / choose not to use the Docker)

These instructions assume a clean RHEL7/CentOS7 server. If any of the following apply to you, please consult SQream Technical Support before installing:

  • your server already has CUDA driver installed
  • you need a different operating system (Ubuntu)
  • you are installing SQream on an IBM Power9 server
